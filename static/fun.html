Appendix B: API Endpoints

Method Endpoint Description Axum Handler
POST /api/auth/register Register a new user with role-based access register_handler
POST /api/auth/login Authenticate user and return JWT token login_handler
GET /api/courses List all courses list_courses_handler
POST /api/courses Create a new course (Teacher only) create_course_handler
GET /api/courses/{id} Retrieve course details get_course_handler
PUT /api/courses/{id} Update course (Teacher only) update_course_handler
DELETE /api/courses/{id} Delete course (Teacher only) delete_course_handler
POST /api/courses/{id}/enroll Enroll student in a course enroll_course_handler
POST /api/courses/{id}/attendance Record attendance for a course (Teacher only) record_attendance_handler
GET /api/courses/{id}/attendance View attendance records for a course get_attendance_handler
POST /api/courses/{id}/assignments Create assignment for a course (Teacher only) create_assignment_handler
GET /api/courses/{id}/assignments List assignments for a course list_assignments_handler
POST /api/assignments/{id}/submit Submit an assignment (Student only) submit_assignment_handler
PUT /api/submissions/{id}/grade Grade a submission (Teacher only) grade_submission_handler
POST /api/forums/{id}/posts Create a new forum post create_forum_post_handler
GET /api/forums/{id}/posts List posts in a forum list_forum_posts_handler
